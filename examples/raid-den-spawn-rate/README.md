[Main Article](https://github.com/necro50n3/cobblemon-raiddens/wiki/Raid-Den-Spawning)

Change the value in `placement.chance` to change the spawn rate of raid dens in the world.
The higher the number, the rarer the raid den.

Add the `data` folder and `pack.mcmeta` into a zip file and add it to the correct `datapacks` folder.
